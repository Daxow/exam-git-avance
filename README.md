## **Présentation :**

Ce projet est un serveur web écrit en Go.

Il permet d’afficher une liste de livres, avec pour chacun l’auteur et l’année de publication.

______________

## **Installation :**

Dans le terminal :

- `go mod init main`

- `go mod tidy`

______________

## **Exécution :**

Dans le terminal :

- `go run main.go`
