## **Présentation :**

Ce projet est un serveur web écrit en Go.

Il permet d’afficher une liste de livres, avec pour chacun l’auteur et l’année de publication.

---------------------

## **Installation :**

Dans le terminal :

Cloner le lien du repo et se placer dans le dossier :

- `git clone <URL_DU_REPO>`
- `cd exam-git-avance`

Puis :

- `go mod init main`
- `go mod tidy`

---------------------

## **Exécution :**

Dans le terminal :

- `go run main.go`
